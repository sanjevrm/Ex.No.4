# 1. Comparative Analysis Prompt
# Definition

A comparative analysis prompt asks for a comparison between two or more concepts by highlighting their similarities and differences.

# Prompt

Compare and contrast smart homes and traditional homes.

# Output

Smart homes use internet-connected devices to automate lighting, security, and appliances, allowing remote control and energy efficiency. Traditional homes rely on manual operation of devices and fixed systems. While smart homes offer convenience, automation, and energy savings, traditional homes are simpler, less expensive, and do not depend on internet connectivity.

# 2. Experiential Perspective Prompt
# Definition

This prompt requests responses from a specific role or personal viewpoint.

# Prompt

As a homeowner, describe your experience using smart home devices.

# Output

As a homeowner, smart home devices make daily life more convenient. I can control lights, fans, and security systems using my smartphone. Automated routines save time and improve safety, though initial setup and technical issues can sometimes be challenging.

# 3. Everyday Functioning Prompt
# Definition

Everyday functioning prompts explain how a system or concept works in daily life.

# Prompt

Describe how smart home technology is used in everyday life.

# Output

Smart home technology is used daily to automatically turn lights on and off, adjust room temperature, monitor security cameras, and control appliances using voice assistants or mobile apps, making homes more efficient and comfortable.

# 4. Universal Prompt Structure (Prompt Refinement)
# Definition

This technique uses a structured template to explain concepts clearly and simply.

# Prompt

Explain smart home technology in simple terms with an example.

# Output

Smart home technology means using the internet to control household devices automatically. For example, a smart light can turn on by itself in the evening, or a user can switch it off remotely using a mobile app.

# 5. Prompt Size Limitation
# Definition

Prompt size limitation manages large prompts by breaking them into smaller tasks.

# Large Prompt

Explain all features and benefits of smart home technology in detail.

# Refined Prompt

Explain five important features of smart home technology first, then explain five benefits separately.

# Large Prompt

List 50 smart home devices.

# Refined Prompt

List 10 commonly used smart home devices.
